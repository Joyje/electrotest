Ett enkelt program som räknar ut ersättningsresistans.

Kör kommandot 'make' för att kompilera programmet eller 'make lib' för att
enbart bygga .so biblioteken.

Efter att programmet kompilerats kan det installeras tillsammans med biblioteken
med 'make install'.
